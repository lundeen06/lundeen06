**What I'm working on:**
* 🛰️ [samwise gnc flight code](https://github.com/stanford-ssi/samwise-adcs-flight) - Attitude Determination & Control System (ADCS) built from the ground up for a 2U Cubesat called SAMWISE. This is a low-cost bus (<10k/unit) with cameras, reaction wheels, a star tracker, S-band comms, and more. In-house design of all subsystems enables us to bring the cost way down. We'll launch in July or October 2027.
